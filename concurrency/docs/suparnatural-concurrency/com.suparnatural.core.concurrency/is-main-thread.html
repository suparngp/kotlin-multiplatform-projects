isMainThread

Returns if the current thread is main thread.

Examples

JobDispatcher.dispatchOnMainThread(Unit) {
    assertTrue(isMainThread())
}

JobDispatcher.dispatchOnBackgroundThread(Unit) {
    assertFalse(isMainThread())
}
