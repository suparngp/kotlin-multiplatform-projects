Future

interface Future<T>

A Future defers the resolution of a value to the future. The value can be consumed by calling Future.consume method. The implementation decides whether Future.consume blocks the calling thread or not.

Functions

consume

 abstract fun <R> consume(code: (T) -> R): R

Consume the future value in code as input. This method may or may not block the calling thread depending upon the implementation.
