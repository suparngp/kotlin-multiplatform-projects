JobDispatcher

expect object JobDispatcher

Dispatches a given job on a particular thread.

Examples

JobDispatcher.dispatchOnMainThread("Hello") {it: String ->
    assertEquals("Hello", it) // runs on main thread
}

JobDispatcher.dispatchOnBackgroundThread("Hello") {it: String ->
    assertEquals("Hello", it) // runs on background thread
}

dispatchOnBackgroundThread

 fun <T> dispatchOnBackgroundThread(payload: T, job: (T) -> Unit): Unit

Dispatches a non-capturing lambda job with payload as its input on a background thread. The payload is converted to an immutable object and then passed to the job as argument.
