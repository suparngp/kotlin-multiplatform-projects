dispatchOnBackgroundThread

fun <T> dispatchOnBackgroundThread(payload: T, job: (T) -> Unit): Unit

Dispatches a non-capturing lambda job with payload as its input on a background thread. The payload is converted to an immutable object and then passed to the job as argument.

On Android, it uses Executor service and on iOS, it uses background queue.
