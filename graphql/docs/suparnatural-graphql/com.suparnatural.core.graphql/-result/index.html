Result

sealed class Result<T, V>

Encapsulates a successful result of type T or a Failure of type V with an arbitrary throwable.

Types

Failure

 data class Failure<T, V> : Result<T, V>

Failure result with a value of type V and a throwable cause. The value can hold details about the failure. For example, http status message. The cause can be null if the failure was not caused by an exception.
