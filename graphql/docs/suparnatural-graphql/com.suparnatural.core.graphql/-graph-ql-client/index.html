GraphQlClient

abstract class GraphQlClient<T>

A client executes operations against a GraphQl server. It accepts a chain of Link which emits the final response of type T.

The client should not concern itself with how to make a network request. Rather, it should rely on the last Link of the chain as terminating link which fetches the response from the server. JsonHttpGraphQlLink is one such link. The client should then take the response as input and transform it into the correct response type based on the operation.
