GraphQlClientFailureResponse(error: GraphQlClientError, response: GraphQlResponse<T>?)

Encapsulates a failed response where response may contain the GraphQl response with GraphQlResponse.errors property set. If the error is not a standard GraphQl error (e.g. network errors), the response will be null.
