split

fun <A, T, V> Link<A, T, V>.split(test: (A) -> Boolean, leftLink: Link<A, *, T>, rightLink: Link<A, *, T>): Link<A, T, V>

Splits the link chain into two branches left and right based on a test. If the test returns true, leftLink chain is followed, otherwise right link chain is followed.
