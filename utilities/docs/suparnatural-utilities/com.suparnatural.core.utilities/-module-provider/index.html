ModuleProvider

abstract class ModuleProvider(ownerIdentifier: String, providerIdentifier: String) : ContentProvider
Content copied to clipboard

A base content provider for auto injecting libraries with context on application startup. The consumer must set the applicationId in android configuration. Otherwise this will raise an exception.

Examples

Injector for My Library

class MyLibraryProvider: ModuleProvider("com.my.library", "myLibraryProvider") {
    override fun bootstrap(context: Context?) {
        FileSystem.initialize(context)
    }
}

Manifest

<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
  package="com.suparnatural.core.fs">
      <application>
          <provider
              android:authorities="${applicationId}.myLibraryProvider"
              android:name=".MyLibraryProvider"
              android:exported="false"
              android:enabled="true"/>
      </application>
</manifest>
